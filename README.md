# 非鱼表情包小程序

## 项目介绍

非鱼表情包小程序是一款专注于提供丰富表情包资源的微信小程序。用户可以浏览、收藏、定制、分享各类有趣的表情包，满足日常社交聊天需求。

## 功能特点

- 🖼️ **丰富表情库**：提供海量表情包资源，分类清晰
- 🔍 **智能搜索**：支持关键词搜索，快速找到想要的表情
- ⭐ **收藏功能**：一键收藏喜欢的表情包，随时使用
- 📤 **便捷分享**：支持一键分享到微信好友
- 🔄 **表情包定制**：定制自己的表情包

## 项目架构
### 前端架构
小程序端：基于 uni-app 开发
管理后台：Vue.js + Element UI，提供完善的内容管理功能，基于若依框架

### 后端架构
基于若依框架

jdk1.8

开发框架：Spring Boot
数据库：MySQL
缓存：Redis
对象存储：本地
